# goslack

`goslack` is a simple go module to send Slack message.

## Installation

```bash
go get -u github.com/ilho-tiger/goslack
```

## Usage

```go
import "github.com/ilho-tiger/goslack"

func main() {
    slack.SendMessage("Hello Slackers!")
}
```
